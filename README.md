# SkilledTalk - Freelance Marketplace

A **professional freelancing platform** where skilled professionals offer services and clients find talent.

## Features

- Freelancer profiles with skills and portfolio
- Advanced search and filtering
- Project posting and proposal system
- In-app messaging
- Secure payment and escrow
- Review and rating system

## Getting Started

```bash
composer install && npm install
cp .env.example .env && php artisan key:generate
php artisan migrate --seed && php artisan serve
```
